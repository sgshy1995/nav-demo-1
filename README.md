# 前端导航站点

个人开发的一个小项目。

## 使用

- JavaScript
- CSS 3
- Vue
- LeanCloud(v1.0.2)
- NodeJs(v1.0.3)

## 功能

**前端导航站点**：

- 完美适配移动端/PC端。(v1.0.1 已实现)
- 自由增/删/改导航地址。(v1.0.1 已实现)
- 可以用四种不同引擎搜索。(v1.0.1 已实现)
- 留言功能。(v1.0.2 开发中)
